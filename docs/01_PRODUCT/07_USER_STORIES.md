# User Stories

---

## Incident Reconstruction

As a SOC Analyst,

I want to upload logs,

so that I can quickly understand the attack timeline.

---

## Threat Memory

As a DFIR Analyst,

I want to find similar investigations,

so that I do not repeat previous work.

---

## Knowledge Builder

As an Incident Responder,

I want every investigation stored as reusable knowledge,

so future analysts benefit.

---

## Knowledge Explorer

As a Threat Hunter,

I want to visualize relationships,

so I can understand attacker behavior.

---

## Similarity Explanation

As an Analyst,

I want ODIN to explain why two incidents are similar,

so I trust the recommendation.

---

## Lessons Learned

As a SOC Manager,

I want every incident to produce reusable lessons,

so my team continuously improves.

---

## Investigation History

As an Analyst,

I want to search historical investigations,

so I can reuse existing playbooks.

---

## Knowledge Search

As a Threat Hunter,

I want semantic search,

so I can find related attacks without knowing exact keywords.
