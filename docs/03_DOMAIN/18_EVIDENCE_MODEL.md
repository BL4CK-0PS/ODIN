# Evidence Model

## Purpose

Evidence is the foundation of trust inside ODIN.

Every recommendation must be backed by evidence.

---

# Evidence Structure

Evidence

├── ID

├── Source

├── Timestamp

├── Description

├── Related Entities

├── Related Techniques

├── Confidence

└── Raw Reference

# Rules

Evidence is immutable.

Evidence cannot be deleted.

Evidence always references original logs.
