# Memory Builder

## Purpose

Convert investigations into reusable institutional memory.

---

# Input

Canonical Incident

---

# Processing

Extract

- Timeline
- Entities
- Evidence
- Techniques
- IOCs
- Playbooks
- Lessons Learned

Normalize

Deduplicate

Link relationships

Generate summary

Create embedding

---

# Output

Memory Object

Memory Object

├── Metadata

├── Canonical Incident

├── Summary

├── Lessons

├── Relationships

├── Embedding

├── Confidence

└── Version

---

# Memory Rules

Memory never replaces previous versions.

Every investigation creates a new version.

Lessons remain immutable.

Evidence remains immutable.

Only metadata may change.
