# ODIN

Operational Defense Intelligence Network

---

## Tagline

Every Investigation Strengthens Tomorrow's Defense.

---

## What is ODIN?

ODIN is an Institutional Cyber Memory platform.

Instead of storing investigations as reports, ODIN transforms every investigation into reusable organizational intelligence.

---

## Features

- Incident Reconstruction
- Threat Memory
- Hybrid Similarity Search
- Investigation Diff
- Knowledge Graph
- Explainable AI
- Local AI Support
